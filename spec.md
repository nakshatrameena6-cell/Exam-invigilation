AI Exam Invigilation is a computer vision-based web application designed to automate online examination monitoring. The system processes uploaded exam videos using YOLO-based object detection and tracking to identify students, monitor their activities, and detect potential exam violations. It provides annotated output videos, violation summaries, and processing reports through a FastAPI backend and a React frontend.

OBJECTIVES:
• Detect and track students throughout the examination using AI.
• Identify suspicious activities such as multiple persons, unauthorized objects (e.g., mobile phones), and abnormal behavior.
• Generate processed videos with real-time annotations and tracking IDs.
• Produce violation summaries and downloadable reports for exam administrators.
• Provide a user-friendly web interface for uploading videos, viewing results, and downloading processed outputs.
• Deliver a scalable, modular, and efficient AI-powered invigilation solution suitable for hackathons and academic demonstrations.
